# miniproject-campsites

> 이 저장소는 전국 야영지 등록 현황을 주제로 서비스 하는 미니 프로젝트를 저장하는 곳입니다.

## 개발 환경

- Front-End : React, TailwindCSS, TW Elements, MUI 등
- Back-End : Spring
- DB : MySQL
